## 快捷键

```
常用
shift + enter ：向下添加一行；
com + enter ：向上添加一行；
com + d ：复制当前行；
com + x	：删除当前行；
alt + shift + ↑/↓ ：移动行，不格式化；
com + shift + ↑/↓ ：移动语句，格式化；
com + shift + -/+	：折叠/展开全部，按多次会有不同效果。
com + -/+	：折叠/展开当前
ctrl + d ：启动；


com + ,	：打开首选项(Preferences)；
```
